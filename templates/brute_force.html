📚 How It Works

● What is a dictionary attack?

An attacker uses a pre-built list of common/leaked passwords and tries each one. The famous RockYou list contains 14 million real passwords from a 2009 breach.

Pseudocode
for password in wordlist: if login(username, password): print(f"Cracked: {password}") break

● Defenses

  • Rate limiting — cap login attempts per IP per minute
  • Account lockout — lock after N consecutive failures
  • CAPTCHA — block automated scripts
  • MFA / 2FA — password alone is never enough
  • Strong passwords — long & random — not in any wordlist
  • Password managers — unique password per service
